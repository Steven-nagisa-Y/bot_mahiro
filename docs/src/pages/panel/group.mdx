# 管理群组

Mahiro 是全白名单设计，所有的群组必须先添加，才能触发消息响应。

## 新增群组

新增群组表单说明：

|字段|说明|
|:-|:-|
|群组名称|必填，用来备注|
|群号|必填|
|过期时间|必填，只有在有效期内才可用|
|绑定账号|需要在该群内响应的 Bot 账号，若一个群有多个 Bot ，请只配置一个 Bot 用来响应即可|
|可用插件|只有配置了的插件，才可以在此群可用，若感觉麻烦，请配置管理员|
|管理员|拥有管理员权限的用户，可以通过命令开关任何已经启用的插件|

### 管理员

管理员可以通过如下命令开关插件：

```bash
  # 开启注册为 `签到` 名称的插件
  .open 签到
  # 关闭 `Hello World` 插件
  .close Hello World
```

<Callout emoji="💡">
  注：只有在 [插件管理](./plugin) 内开启的插件，才可以被管理。
</Callout>
