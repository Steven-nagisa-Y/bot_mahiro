# 管理插件

## 插件注册

插件是自动注册，来源主要有以下几类：

|来源|说明|
|:-:|:-|
|内置|内部插件，包括 Python Bridge 转发插件，Admin 群组管理员插件，不可关闭|
|用户注册|用户通过 `mahiro.onGroupMessage` 注册的插件|
|Python Bridge|若使用了 [Python Bridge](../python/bridge) ，Python 插件也将注册进来|

## 插件开关

所有 非内置 插件可全局关闭，在此处关闭后，所有群都不可用，群管理员也无法开启。

## 插件黑白名单

### 白名单用户

是一种全局特权用户。

被全局关闭的插件，仍然可以被白名单用户无条件触发，即使某个群没有配置该插件，该白名单用户仍然可以使用。

### 黑名单用户

是一种全局屏蔽用户。

被屏蔽的用户在任何场景下都无法使用该插件。
